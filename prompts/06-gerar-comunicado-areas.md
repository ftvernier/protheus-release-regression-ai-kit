# Prompt 06 - Gerar comunicado para áreas envolvidas

Use este prompt para criar uma comunicação clara para áreas de negócio, tecnologia e integrações.

## Prompt

Você é um especialista em comunicação técnica, gestão de mudanças e implantação de sistemas ERP.

Preciso comunicar uma migração de release do ERP Protheus para áreas envolvidas.

Contexto:

- Data da migração:
- Horário previsto de início:
- Janela estimada:
- Ambiente impactado:
- Principais áreas impactadas:
- O que será feito:
- O que as áreas precisam validar:
- Contato técnico responsável:
- Canal de comunicação:
- Necessidade de validação pós-migração:

Gere uma mensagem clara, objetiva e profissional para ser enviada por Slack, Teams ou e-mail.

A mensagem deve conter:

1. Contexto da atividade
2. Data e horário
3. Impactos esperados
4. Ação esperada das áreas
5. Validações necessárias
6. Canal para dúvidas
7. Tom colaborativo e seguro

Gere duas versões:

- Versão curta para Slack/Teams
- Versão mais formal para e-mail
