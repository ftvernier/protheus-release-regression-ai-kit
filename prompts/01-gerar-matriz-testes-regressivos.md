# Prompt 01 - Gerar matriz de testes regressivos

Use este prompt para gerar uma matriz inicial de testes regressivos para uma migração de release do ERP Protheus.

## Prompt

Você é um especialista em ERP Protheus, testes regressivos, sustentação de sistemas críticos e migração de release.

Estou conduzindo uma migração de release do ERP Protheus.

Contexto da migração:

- Release atual:
- Release destino:
- Banco de dados:
- Sistema operacional:
- Ambiente de aplicação:
- Principais módulos utilizados:
- Principais integrações:
- Principais customizações:
- Processos críticos da empresa:
- Data prevista para go-live:
- Janela estimada de migração:

Com base nesse contexto, gere uma matriz de testes regressivos contendo:

1. Módulo
2. Processo de negócio
3. Cenário de teste
4. Objetivo do teste
5. Pré-condições
6. Passos para execução
7. Resultado esperado
8. Evidência necessária
9. Criticidade
10. Responsável sugerido
11. Status
12. Observações

Classifique a criticidade em:

- Alta
- Média
- Baixa

Dê prioridade para processos que possam impactar faturamento, estoque, fiscal, financeiro, integrações, emissão de notas fiscais, pedidos de venda, compras, rotinas automáticas, APIs, jobs e customizações.

Gere a saída em formato de tabela Markdown.
