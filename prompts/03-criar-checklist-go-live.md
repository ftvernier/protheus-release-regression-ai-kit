# Prompt 03 - Criar checklist de go-live

Use este prompt para gerar um checklist de go-live para migração de release do ERP Protheus.

## Prompt

Você é um especialista em ERP Protheus, sustentação de ambientes críticos, implantação de sistemas e gestão de go-live.

Estou preparando o go-live de uma migração de release do ERP Protheus.

Contexto:

- Release atual:
- Release destino:
- Ambiente de produção:
- Banco de dados:
- Sistema operacional:
- Principais serviços Protheus:
- Integrações críticas:
- Rotinas automáticas:
- TSS utilizado:
- APIs utilizadas:
- Janela de indisponibilidade:
- Áreas envolvidas:
- Plano de rollback:

Gere um checklist de go-live dividido em:

1. Antes da migração
2. Durante a migração
3. Após a migração
4. Validação funcional
5. Validação técnica
6. Validação de integrações
7. Validação fiscal
8. Comunicação com áreas de negócio
9. Monitoramento pós-go-live
10. Critérios para rollback

Para cada item, inclua:

- Etapa
- Atividade
- Responsável sugerido
- Criticidade
- Evidência esperada
- Status
- Observações

Gere a saída em formato de tabela Markdown.
